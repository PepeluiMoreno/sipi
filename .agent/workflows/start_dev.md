# Start SIPI Frontend

This workflow starts the Vue 3 frontend development server.

1. Ensure you are in the `sipi-frontend` directory.
2. Install dependencies if needed:
   ```powershell
   npm install
   ```
3. Start the development server:
   // turbo
   ```powershell
   npm run dev
   ```
4. The application will serve at `http://localhost:5173` (or similar port).
